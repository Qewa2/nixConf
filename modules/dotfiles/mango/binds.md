# Mango Keybinds

Super, w -> open this help window

Super, r -> reloar the Configuration at runtime

Alt, Enter -> open kitty

Alt, Space -> open rofi
Alt, Shift, Space -> open wofi

Super, Escape -> Quit mango

Alt, c -> Kill current App

## Windows
### Focusing
Alt, Tab -> focus next App

Alt, h/j/k/l -> focus left/down/up/right (vim motions)

Alt, Shift, h/j/k/l -> swap window with the left/down/up/right window

### Resize
Alt, Ctrl, h/j/k/l -> resize window (only floating)

### Window Modes
Super, o -> open Overview

Super, g -> make window visible everywhere

### Fullscreening
Super, f -> makes window fullscreen

Super, ctrl, f -> makes window as big as it can be (1.0 width)

### Minimizing
Super, i -> minimize window

Super, Shift, i -> make minimizer window reapper

### Switching Layouts
Super, n -> cycles through configured Layouts

Super, Shift, c -> switches to scroller

## Worspaces/Tags
### Focusing
Super, 1,2,3,4,5,6,7,8,9,0 -> focus workspace 1,2,3,4,...

Super, h/l -> view the left/right workspace if it has an open window

### Moving to Workspace
Super, Shift, 1,2,3,4,5,6,7,8,9,0 -> move focused window to workspace 1,2,3,4,...

Super, Shift, h/l -> move focused window to the left/right workspace

## Monitors
### Focusing
Super, Ctrl, h/j/k/l -> focus the monitor to the left/down/up/right

### Moving
SUPER, Alt, h/j/k/l -> move focused window the the left/down/up/right monitor

## gaps
Alt, g -> toggle gaps
Alt, + -> inrease gaps
Alt, - -> decrease gaps
